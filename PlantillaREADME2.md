
# 📊 Finance Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Finance Dashboard** built using **Microsoft Power BI** to analyze personal financial data including **Income, Expenses, and Savings** across multiple years.

The dashboard provides insights into financial trends, category-wise spending, and savings behavior to support better financial decision-making.

---

## ❗ Problem Statement

Managing personal finances effectively is a common challenge. Individuals often struggle to:

* Track their **income, expenses, and savings** in a structured way
* Identify **where money is being spent the most**
* Analyze **financial trends over time**
* Maintain a **balance between expenses and savings**
* Make **data-driven financial decisions**

Due to lack of proper visualization and analysis tools, financial data remains scattered and underutilized.

👉 This project aims to solve these challenges by building an **interactive and visually rich dashboard** that transforms raw financial data into meaningful insights.

---

## 🚀 Features

* 📅 **Dynamic Date & Year Filters** (2021–2024)
* 💰 **Income, Expense & Savings Summary Cards**
* 📈 **Expense Trend Analysis (Year-wise)**
* 🥧 **Category-wise Expense & Savings Distribution**
* 📊 **Detailed Expense Breakdown Table**
* 🎯 **Savings Percentage & Expense vs Savings Comparison**
* 🔍 Interactive slicers and filters for deep analysis

---

## 📊 Key Insights

* 📉 Expenses show a declining trend after 2022
* 🏠 **House Rent** is the highest expense category
* 📊 Savings rate is approximately **63%**
* ⚖️ Expense vs Savings ratio indicates controlled spending
