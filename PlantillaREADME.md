# Personal Finance Dashboard | Excel Interactive Dashboard

## 📌 Project Overview

Designed and developed an interactive **Personal Finance Dashboard in Microsoft Excel** to track and analyze income, expenses, investments, budgets, and payment methods.
The dashboard provides real-time financial insights using Pivot Tables, Pivot Charts, Slicers, KPIs, and advanced Excel visualization techniques.

---

## 🎯 Problem Statement

Managing personal finances manually makes it difficult to:

* Track monthly spending patterns
* Monitor investments and balances
* Analyze payment method usage
* Compare income vs expenses
* Make data-driven financial decisions

This dashboard solves these challenges through a dynamic and visually interactive finance analytics system.

* Credit Card and UPI are the most frequently used payment methods.
* Salary contributes the highest share of income.
* Expenses peak during mid-year months.
* Investment trends remain stable throughout the year.
* Some locations show significantly higher spending patterns.
